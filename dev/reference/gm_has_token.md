# Is there a token on hand?

Reports whether gmailr has stored a token, ready for use in downstream
requests.

## Usage

``` r
gm_has_token()
```

## Value

Logical.

## See also

Other low-level API functions:
[`gm_token()`](https://gmailr.r-lib.org/dev/reference/gm_token.md)

## Examples

``` r
gm_has_token()
#> [1] FALSE
```
