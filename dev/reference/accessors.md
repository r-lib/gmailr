# Methods to get values from message or drafts

Methods to get values from message or drafts

## Usage

``` r
gm_to(x, ...)

gm_from(x, ...)

gm_cc(x, ...)

gm_bcc(x, ...)

gm_date(x, ...)

gm_subject(x, ...)
```

## Arguments

- x:

  the object from which to get or set the field

- ...:

  other parameters passed to methods
