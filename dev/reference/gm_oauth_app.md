# Get currently configured OAuth app (deprecated)

**\[deprecated\]**

In light of the new
[`gargle::gargle_oauth_client()`](https://gargle.r-lib.org/reference/gargle_oauth_client_from_json.html)
constructor and class of the same name, `gm_oauth_app()` is being
replaced by
[`gm_oauth_client()`](https://gmailr.r-lib.org/dev/reference/gm_auth_configure.md).

## Usage

``` r
gm_oauth_app()
```
