# Get the body text of a message or draft

Get the body text of a message or draft

## Usage

``` r
gm_body(x, ...)
```

## Arguments

- x:

  the object from which to retrieve the body

- ...:

  other parameters passed to methods

## Examples

``` r
if (FALSE) { # \dontrun{
gm_body(my_message)
gm_body(my_draft)
} # }
```
